
# OpenGL Template.

OpenGL template for quick set up of 2D OpenGL projects -- despite this is very easy to set up a 3D one just changing some of the structure.

It is aimed for my personal use, but any improvement requested by any user would be wonderfuly welcome.

Moreover, it is likely going to be updated as I advance in the development of https://github.com/nobody-but-me/bug-game.git since that is the source/origin project of this template.

# Dependencies:

 - cmake 3.21 or higher;
 - clang and clang++;
 - ninja;
 - glfw3;
 - ;

# Building:

```bash

 > sh build.sh

```

And follow the instructions.
